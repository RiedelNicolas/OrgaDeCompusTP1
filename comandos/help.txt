Comandos:
Ejemplos de uso:
ejecutable -h
ejecutable -V
ejecutable -i in_file -o out_file


-V, --version
Imprime la version del programa y finaliza la ejecucion.
-h, --help
Muestra esto por pantalla y finaliza la ejecucion.
-i, --input
Especificar la ruta para el archivo de entrada, "-" para stdin.
-o, --output
Especificar la ruta para el archivo de salida, "-" para stdout.
